# Hxmaker
Hxmaker是一个仅负责处理显示对象列表的游戏引擎，它可以提供常见引擎所需要的渲染列表数据。
这意味着它不依赖任何引擎，可以根据自已的实际需求，设计新的游戏引擎后端。

# 支持引擎后端
1. [OpenFL](https://github.com/rainyt/hxmaker-openfl)

# 支持
1. 基础显示对象（图片、文本、图形绘制等）
2. 基础组件（按钮、Scroll等）
3. 基础布局（垂直布局、水平布局等）
4. 音频
5. 资源管理
